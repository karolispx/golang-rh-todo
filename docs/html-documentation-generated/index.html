Up 
get /tasks
Get user tasks. (tasksGet)
Get user tasks by query parameters.

Query parameters

watching (optional)
Query Parameter — Bring back all tasks that are being watched or not watched. Default null (will bring back all), available: yes, no.
limit (optional)
Query Parameter — The numbers of items to return, default 10.
offset (optional)
Query Parameter — The numbers of items to skip from the beginning, default 0.
orderby (optional)
Query Parameter — Order results by, default taskid, available: taskid, task, date_created, date_updated.
order (optional)
Query Parameter — Order results coming back in, default desc, available: asc, desc.

Responses

200

Tasks returned

403

Not authorized to access this area

422

Bad details, bad payload, missing details, cooldown period

500

Server error
